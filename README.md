# 💰 Loan EMI Calculator

A Python-based Loan EMI Calculator that computes monthly installments,
generates a full amortization schedule, and visualizes repayment data.

## Features
- Calculates monthly EMI using standard amortization formula
- Generates month-by-month repayment schedule
- Exports schedule to CSV file
- Visualizes balance and interest breakdown using Matplotlib

## How to Run

1. Install dependencies:
   pip install pandas matplotlib

2. Run the program:
   python emi_calculator.py

3. Enter loan amount, interest rate, and tenure when prompted

## Technologies Used
- Python 3
- Pandas
- Matplotlib

## Sample Output
- Monthly EMI, Total Payment, Total Interest printed to console
- amortization_schedule.csv generated in project folder
- Loan repayment chart displayed and saved as PNG
